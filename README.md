## 📌 About the project

**Gás Já!** is a digital platform designed to connect gas suppliers with customers in a fast and efficient way.

The goal is to modernize a process that is still very manual in Brazil ordering gas by bringing a user experience similar to delivery apps.

---

## 🎯 Problem

Currently, ordering gas usually requires users to:
- call suppliers manually  
- search for contacts  
- deal with delivery uncertainty  

---

## 💡 Solution

Gas Já! solves this by providing:

- simple user registration (customers and suppliers)  
- automatic location detection  
- connection with nearby suppliers  
- a fully digital flow  

---

## 🚀 Features

### 🌍 Geolocation
- Automatic user location detection  
- Displays neighborhood + state  

### 📍 Smart Address System
- Address auto-fill via ZIP code (ViaCEP API)  
- Automatically fills:
  - street  
  - neighborhood  
  - city  
  - state  

### 🧠 Advanced Validation
- Real-time form validation  
- Visual feedback (error / success)  
- Valid CNPJ verification (real algorithm)  
- Password rules (length, uppercase, number)  

### 📱 User Experience (UX)
- Input masks (phone, ZIP code, CNPJ)  
- Responsive interface  
- Smooth multi-step navigation  

### 🔐 Input Security
- Show/hide password  
- Prevent invalid form submission  

---

## 🔄 User Flow

1. User accesses the platform  
2. Chooses:
   - Order gas  
   - Become a supplier  
3. Completes registration  
4. Adds address  
5. (Next step) View nearby suppliers
